Introduction to CSS



What is CSS?

Css is the short form of Cascading Styles Sheets. It provides visual styling and layout for everything on a webpage i.e. with the help of css you can change the text's size,color,font, add background image to website etc. We can make a simple HTML page look good with the help of css. We can add simple animations and interactions through css for example: a button’s color changes when we hover on it thus giving a clickable effect.


Types of CSS

There are three types of css through which we can link it to the Html page which are as follows:

1. External Css: This contains a separate CSS file saved with ‘.css’ extension. This file can be applied to the html document by referencing it within the head tag of the html document with the help of 'link' element. This is generally used when we want to apply same styles to multiple html pages.

<head>
<link rel="stylesheet" href="styles.css">
</head>

Here in this link element there are two attributes which are 'rel' and 'href'. The rel attribute tells the browser what relationship it has with the linked document.

Here rel="stylesheet" tells the browser that we are importing a stylesheet.

The 'href' attribute contains the path to our css file which has styles we want to apply to our html file.

2. Inline Css: This uses the style attribute which is added to the opening of the html tag. The Css style rules are added as value to the style attribute which then applies the style directly to the content which is mentioned between this opening and closing html tag. This is normally used when you want to apply a particular style to a particular html element only.

<h1 style="color:green"> Heading </h1>

In the above example heading font color would be green.

3. Internal Css: Here the Css is added within the ‘style’ tag within the head of the html document.

<head>
<style>
p{
color:blue;
}
</style> </head>

In the above example the text color of the paragraph would be blue.


Let's Revise

So in short Css is a way we can beautify our simple html pages. To use Css with Html we can either use a external file by linking it with our html page or we can add styles to our style tag within the head of html document. Inline css can also be used to add css to our html document but it is most preferred when we want to add a different style to an element.
