Introduction to CSS Selectors



What are CSS Selectors?

Css Selectors are used to apply styles to the html element you want to. Css Selectors select Html elements according to its id, class, type, attribute etc


Types of CSS Selectors

The different types of Css selectors are as follows:

1. Element Selector: This is the most basic kind of selectors which matches the particular style to the particular html element mentioned.

h1{
color:red;
}

This will turn the text color to red of all the elements belonging to the h1 tag of the html document.

2. Universal Selector: This applies styles to all the elements of the html document irrespective of which type,class or id they belong to the html document. It is denoted by ‘*’ .

*{
text-align:center;
}

This will align all the contents of the html document to the center of the web page.

3. Class Selector: This applies style to the class attribute of the html document. To select elements with the specific class we use the ‘.’ symbol at the starting followed by the class name to which we want to apply styles.

<p class=”fancy”> Fancy Paragraph</p>
.fancy{
font-style:italic;
font-weight:bold;
}

This will make the contents of the fancy class italic and bold.

4. Id selector: This applies the style to the id attribute of the html document. To select elements with the specific id we use the ‘#’ symbol at the starting followed by the id name to which we want to apply styles.

<section id=”#sections”>
<p> Paragraph 1 </p>
<p> Paragraph 1 </p>
</section>
#sections{
color:red
}

This would change the color of content of id sections to red.

5.Grouping Selector: This selector is used to make a group of the html elements which have the same style definitions. To group selectors we separate each selector by a ‘,’.

h1,h2,p{
color:green;
text-align:center;
}

This would change the color of the contents within the h1,h2 and p tag to green and align them to center.

Selector Example Example Description
#sections Selects the element with the id="sections"
.fancy Selects the element with the class="sections"
* Selects all the elements
h1 Selects all the <h1> element
